# Daily Answer Sync Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Automatically fetch confirmed Wordle answers from the NYT API daily via GitHub Actions, replacing the stale prediction-based word list.

**Architecture:** A new Node.js script fetches answers from `nytimes.com/svc/wordle/v2/{date}.json` for all dates from origin (2021-06-19) through yesterday, writes them to the existing data files, and regenerates derived TypeScript. A GitHub Action runs this daily after the 5:00 AM UTC Wordle rollover.

**Tech Stack:** Node.js (ESM script), GitHub Actions, NYT public Wordle API

### Task 1: Create the fetch-answers script

**Files:**
- Create: `scripts/fetch-answers.mjs`

**Step 1: Write the fetch script**

This script:
- Fetches every answer from origin through yesterday using the NYT API
- Writes the chronological comma-separated list to `src/data/answers_by_date_wordlist.txt`
- Regenerates `src/lib/wordle/answersByDate.ts` from that list
- Is idempotent — safe to re-run any time

```js
#!/usr/bin/env node
import fs from 'node:fs';
import path from 'node:path';

const ORIGIN = new Date('2021-06-19');
const API_BASE = 'https://www.nytimes.com/svc/wordle/v2';

const root = process.cwd();
const dataDir = path.join(root, 'src', 'data');
const answersByDateTxtPath = path.join(dataDir, 'answers_by_date_wordlist.txt');
const answersByDateTsPath = path.join(root, 'src', 'lib', 'wordle', 'answersByDate.ts');

function formatDate(d) {
  const yyyy = d.getUTCFullYear();
  const mm = String(d.getUTCMonth() + 1).padStart(2, '0');
  const dd = String(d.getUTCDate()).padStart(2, '0');
  return `${yyyy}-${mm}-${dd}`;
}

function yesterdayUtc() {
  const now = new Date();
  const y = new Date(Date.UTC(now.getUTCFullYear(), now.getUTCMonth(), now.getUTCDate()));
  y.setUTCDate(y.getUTCDate() - 1);
  return y;
}

async function fetchAnswer(dateStr) {
  const url = `${API_BASE}/${dateStr}.json`;
  const res = await fetch(url);
  if (!res.ok) throw new Error(`HTTP ${res.status} for ${dateStr}`);
  const json = await res.json();
  if (typeof json.solution !== 'string' || json.solution.length !== 5) {
    throw new Error(`Invalid solution for ${dateStr}: ${JSON.stringify(json.solution)}`);
  }
  return json.solution.toLowerCase();
}

async function main() {
  const endDate = yesterdayUtc();
  const dates = [];
  const d = new Date(ORIGIN);
  while (d <= endDate) {
    dates.push(formatDate(d));
    d.setUTCDate(d.getUTCDate() + 1);
  }

  console.log(`Fetching ${dates.length} answers (${dates[0]} to ${dates[dates.length - 1]})...`);

  // Read existing answers to avoid re-fetching unchanged ones.
  let existing = [];
  try {
    const raw = fs.readFileSync(answersByDateTxtPath, 'utf8');
    existing = raw.split(/[\r\n,]+/).map(w => w.trim().toLowerCase()).filter(w => /^[a-z]{5}$/.test(w));
  } catch { /* first run */ }

  const answers = [];
  let fetched = 0;
  for (let i = 0; i < dates.length; i++) {
    if (i < existing.length) {
      // Trust existing entry — only re-fetch if we need more entries.
      answers.push(existing[i]);
      continue;
    }
    const word = await fetchAnswer(dates[i]);
    answers.push(word);
    fetched++;
    // Rate-limit: small delay between API calls.
    if (fetched % 50 === 0) {
      console.log(`  ...fetched ${fetched} new answers (up to ${dates[i]})`);
      await new Promise(r => setTimeout(r, 200));
    }
  }

  console.log(`Done. ${answers.length} total answers, ${fetched} newly fetched.`);

  // Write answers_by_date_wordlist.txt (comma-separated, single line).
  fs.writeFileSync(answersByDateTxtPath, answers.join(',') + '\n', 'utf8');

  // Write answersByDate.ts.
  const tsContent = `// Auto-generated by scripts/fetch-answers.mjs — do not edit manually
// originDate: 2021-06-19 (Wordle #1 = cigar)
export const ANSWERS_BY_DATE: string[] = ${JSON.stringify(answers)};
`;
  fs.writeFileSync(answersByDateTsPath, tsContent, 'utf8');

  console.log(`Wrote ${answersByDateTxtPath}`);
  console.log(`Wrote ${answersByDateTsPath}`);
}

main().catch(err => {
  console.error(err);
  process.exit(1);
});
```

### Task 3: Create the GitHub Actions workflow

**Files:**
- Create: `.github/workflows/update-answers.yml`

**Step 1: Write the workflow file**

```yaml
name: Update Wordle Answers

on:
  schedule:
    # Run daily at 5:30 AM UTC (30 min after Wordle rollover)
    - cron: '30 5 * * *'
  workflow_dispatch: # Allow manual trigger

permissions:
  contents: write

jobs:
  update-answers:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - uses: actions/setup-node@v4
        with:
          node-version: '20'

      - name: Install dependencies
        run: npm ci

      - name: Fetch latest answers from NYT
        run: npm run fetch:answers

      - name: Regenerate wordlists
        run: npm run sync:wordlists

      - name: Commit and push if changed
        run: |
          git config user.name "github-actions[bot]"
          git config user.email "github-actions[bot]@users.noreply.github.com"
          git diff --quiet && echo "No changes" && exit 0
          git add src/data/ src/lib/wordlists.ts src/lib/wordle/answersByDate.ts
          git commit -m "chore: update Wordle answers $(date -u +%Y-%m-%d)"
          git push
```

**Step 2: Commit**

```bash
git add .github/workflows/update-answers.yml
git commit -m "ci: add daily GitHub Action to sync Wordle answers from NYT API"
```

### Task 4: Write a test for the fetch-answers script's date logic

**Files:**
- Create: `scripts/fetch-answers.test.mjs`

**Step 1: Write the test**

We can't test the full fetch in CI (rate limits), but we can validate the date math and file generation logic. Extract and test the key helpers.

```js
import assert from 'node:assert/strict';
import { test } from 'node:test';

// Inline the pure functions we want to test.
function formatDate(d) {
  const yyyy = d.getUTCFullYear();
  const mm = String(d.getUTCMonth() + 1).padStart(2, '0');
  const dd = String(d.getUTCDate()).padStart(2, '0');
  return `${yyyy}-${mm}-${dd}`;
}

test('formatDate produces YYYY-MM-DD', () => {
  assert.equal(formatDate(new Date('2021-06-19T00:00:00Z')), '2021-06-19');
  assert.equal(formatDate(new Date('2026-03-05T12:00:00Z')), '2026-03-05');
});

test('origin date is Wordle #1', () => {
  // Wordle #1 = cigar = 2021-06-19
  assert.equal(formatDate(new Date('2021-06-19')), '2021-06-19');
});

test('date range covers expected number of days', () => {
  const origin = new Date('2021-06-19');
  const end = new Date('2026-03-04'); // yesterday as of plan date
  const days = Math.floor((end - origin) / (1000 * 60 * 60 * 24));
  // Should be 1719 days, meaning 1720 entries (inclusive of origin)
  assert.equal(days + 1, 1720);
});
```
